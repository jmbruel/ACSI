::Q1:: Que signifie MCT?
{       =Modèle Conceptuel de Traitement # right; bonne réponse!
        ~Modèle Computationel de Traitement # wrong, faux !
        ~Méthode Conceptuel de Traitement# wrong, faux !
}

::Q2:: Quel élément du MCT se rapproche le plus des flux du MCF ?
{   ~action
    ~flux
	=événement
    ~résultat
}

::Q3:: A quoi doit répondre un MCT ?
{       
	~Qui ? # wrong, Quoi ?
	=Quoi ? # right; good!
	~Comment ? # wrong, Quoi?
	~A qui ? # wrong, Quoi? 
}

::Q4:: Qu'est-ce qui peut séparer une opération dans un MCT ?
{       =Un flux venant d'un acteur externe # right; good!
        ~Un flux vers un acteur externe # wrong, Un flux venant d'un acteur externe
        ~rien # wrong, Un flux venant d'un acteur externe 
}
